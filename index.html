    MediaEase Documentation¶

    Welcome to the MediaEase documentation. This guide provides comprehensive details on setting up and managing MediaEase and HarmonyUI. HarmonyUI is a versatile web interface designed for managing systemd services. It can operate independently or as part of the full MediaEase system. MediaEase, when installed with its accompanying scripts, is tailored to manage only those services and applications that are part of its ecosystem.

    Introduction¶

    Understanding the distinction between HarmonyUI and MediaEase is key to utilizing their full potential.

    • About HarmonyUI – Learn about HarmonyUI as a standalone web interface for managing systemd services.
    • About MediaEase – Discover how MediaEase manages services and applications within its own environment.

    Getting Started¶

    Begin your journey with detailed setup instructions for either HarmonyUI or the full MediaEase system.

    • HarmonyUI Standalone Installation – Step-by-step instructions to install only HarmonyUI.
    • MediaEase Full Installation – Comprehensive guide to install the full MediaEase project, including HarmonyUI and scripts.
    • Configuration – Configuration tips for both HarmonyUI and MediaEase.

    MediaEase Modules¶

    Deep dive into the modules of MediaEase and explore how HarmonyUI integrates within the system.

    • Scripts – Details on MediaEase scripts for task automation within the MediaEase ecosystem.
    • HarmonyUI Integration – Insights into how HarmonyUI functions within the MediaEase environment.
    • MediaEase Core – An overview of the core functionalities of the MediaEase system.

    Community and Support¶

    Engage with the community and access support resources for both HarmonyUI and MediaEase.

    • Discord Channel – Connect with the user and developer community on Discord for real-time discussions and support.
    • GitHub Project – Access the GitHub project to participate in discussions, track progress, and contribute to the MediaEase ecosystem.

    Licensing¶

    Understand the licensing details for both HarmonyUI and MediaEase.

    • HarmonyUI License – Licensing information for HarmonyUI as a standalone product.
    • MediaEase License – Details on the licensing for the full MediaEase system.
